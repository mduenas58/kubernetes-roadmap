# Kubernetes Learning Roadmap

> A structured, self-paced roadmap to go from zero to Kubernetes master — using GitHub for version control and Obsidian for notes.

## How to Use This Repo

1. **Clone** this repository and open it as an Obsidian vault.
2. Start at `00-Prerequisites/` and work your way through each phase.
3. Use the `Progress/Tracker.md` to track completed topics.
4. Do the labs in `Labs/` as you finish each phase.
5. Commit your notes and lab work to keep a learning journal.

## Folder Structure

```
kubernetes-roadmap/
├── 00-Prerequisites/       Linux, Docker, Networking, YAML
├── 01-Core-Concepts/       Architecture, Pods, Workloads, Services
├── 02-Intermediate/        Storage, RBAC, Helm, Resource Management
├── 03-Advanced/            Cluster setup, Observability, Service Mesh, Operators
├── 04-Master/              GitOps, Security, Multi-cluster, Internals
├── Labs/                   Hands-on exercises by phase
├── Resources/              Books, courses, certifications
└── Progress/               Personal tracker
```

## Obsidian Setup Tips

- Install the **Dataview** plugin to make `Progress/Tracker.md` work.
- Install the **Calendar** plugin to journal daily study sessions.
- Enable **Graph View** to visualize how topics connect.
- Use tags like `#kubernetes`, `#status/done`, `#status/in-progress` to filter notes.
