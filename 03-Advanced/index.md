## Time Estimate

- 6–8 weeks at 2 hours/day

## Phase Exit Checklist

- [ ] Can bootstrap a Kubernetes cluster from scratch using kubeadm
- [ ] Have Prometheus + Grafana running, with custom dashboards and alerts
- [ ] Deployed an application via a CI/CD pipeline (not manually)
- [ ] Installed and configured a service mesh for mTLS and traffic management
- [ ] Written a CRD and a basic operator/controller
- [ ] Ready to sit CKS exam
