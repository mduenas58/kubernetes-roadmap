## Phase Exit Checklist

- [ ] Can deploy a stateful app (database) with persistent storage
- [ ] Can create Roles, ClusterRoles, and RoleBindings for least-privilege access
- [ ] Can write and enforce Network Policies for namespace isolation
- [ ] Can set resource requests/limits and configure HPA
- [ ] Can install, customize, and upgrade a Helm chart
- [ ] Ready to sit CKA or CKAD exam
